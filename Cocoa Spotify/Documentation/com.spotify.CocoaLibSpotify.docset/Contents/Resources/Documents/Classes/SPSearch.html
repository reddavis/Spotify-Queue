Overview

This class performs a search on the Spotify catalogue available to the given session, returning tracks, albums and artists.

Class Methods

liveSearchWithSearchQuery:inSession:

Creates a new “live search” SPSearch with the default page size for the given query.

+ (SPSearch *)liveSearchWithSearchQuery:(NSString *)searchQuery inSession:(SPSession *)aSession

Parameters

searchQuery

The search query to create an SPSearch for.

aSession

The SPSession the track should exist in.

Return Value

Returns the created SPSearch object.

Discussion

Live searches should be used when displaying “live search” UI — for example, a search suggestion UI as the user is typing into a search field. This convenience method is simply returns a new, autoreleased SPSearch object. No caching is performed.

Warning: The search query will be sent to the Spotify search service immediately. Be sure you want to perform the search before creating the object!
