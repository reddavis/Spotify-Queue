Class Methods

imageWithImageId:inSession:

Creates an SPImage from the given ID.

+ (SPImage *)imageWithImageId:(const byte *)imageId inSession:(SPSession *)aSession

Parameters

imageId

The image ID to create an SPImage for.

aSession

The SPSession the image should exist in.

Return Value

Returns the created SPImage object.

Discussion

This convenience method creates an SPImage object if one doesn’t exist, or returns a cached SPImage if one already exists for the given ID.

Warning: This method must be called on the libSpotify queue. See the “Threading” section of the library’s readme for more information.

Return Value

Discussion

Warning: This method must be called on the libSpotify queue. See the “Threading” section of the library’s readme for more information.

Warning: For better performance and built-in caching, it is recommended you create SPImage objects using [SPImage imageWithImageId:inSession:].
