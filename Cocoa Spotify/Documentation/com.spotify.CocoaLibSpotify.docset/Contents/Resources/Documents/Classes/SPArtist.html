Overview

Represents an artist on Spotify.

SPArtist is roughly analogous to the sp_artist struct in the C LibSpotify API.

artistWithArtistStruct:inSession:

Creates an SPArtist from the given opaque sp_artist struct.

+ (SPArtist *)artistWithArtistStruct:(sp_artist *)anArtist inSession:(SPSession *)aSession

Parameters

anArtist

The sp_artist struct to create an SPArtist for.

aSession

The session to create the artist in.

Return Value

Returns the created SPArtist object.

Discussion

This convenience method creates an SPArtist object if one doesn’t exist, or returns a cached SPArtist if one already exists for the given struct.

Warning: This method must be called on the libSpotify queue. See the “Threading” section of the library’s readme for more information.

artistWithArtistURL:inSession:callback:

Creates an SPArtist from the given Spotify artist URL.

+ (void)artistWithArtistURL:(NSURL *)aURL inSession:(SPSession *)aSession callback:(void ( ^ ) ( SPArtist *artist ))block

Parameters

aURL

The artist URL to create an SPArtist for.

aSession

The session to create the artist in.

block

The block to be called with the created SPArtist object, or nil if given an invalid artist URL.

Discussion

This convenience method creates an SPArtist object if one doesn’t exist, or returns a cached SPArtist if one already exists for the given URL.

Warning: If you pass in an invalid artist URL (i.e., any URL not starting spotify:artist:, this method will return nil.
