folderId

Returns the folder’s ID, as used in the C LibSpotify API.

@property (nonatomic, readonly) sp_uint64 folderId

Discussion

Warning: This should only be used if you plan to directly use the C LibSpotify API. The behaviour of CocoaLibSpotify is undefined if you use the C API directly on items that have CocoaLibSpotify objects associated with them.
