Overview

Represents an album on Spotify.

SPAlbum is roughly analogous to the sp_album struct in the C LibSpotify API.

Class Methods

albumWithAlbumStruct:inSession:

Creates an SPAlbum from the given opaque sp_album struct.

+ (SPAlbum *)albumWithAlbumStruct:(sp_album *)anAlbum inSession:(SPSession *)aSession

Parameters

anAlbum

The sp_album struct to create an SPAlbum for.

aSession

The SPSession the album should exist in.

Return Value

Returns the created SPAlbum object.

Discussion

This convenience method creates an SPAlbum object if one doesn’t exist, or returns a cached SPAlbum if one already exists for the given struct.

Warning: This method must be called on the libSpotify queue. See the “Threading” section of the library’s readme for more information.

albumWithAlbumURL:inSession:callback:

Creates an SPAlbum from the given Spotify album URL.

+ (void)albumWithAlbumURL:(NSURL *)aURL inSession:(SPSession *)aSession callback:(void ( ^ ) ( SPAlbum *album ))block

Parameters

aURL

The album URL to create an SPAlbum for.

aSession

The SPSession the album should exist in.

block

Block to be called with the created SPAlbum object, or nil if given an invalid album URL.

Discussion

This convenience method creates an SPAlbum object if one doesn’t exist, or returns a cached SPAlbum if one already exists for the given URL.

Warning: If you pass in an invalid album URL (i.e., any URL not starting spotify:album:, this method will return nil.
